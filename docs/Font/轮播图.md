### 实现轮播图

* 前提准备
轮播图容器HTML

``` HTML
 <div class="container">
  <div class="imglist">
   <li>存放图片的标签</li>>
   <li>存放图片</li>
  </div>
  <ul class="wrap-bar">
                <li data-id="0">0</li>
                <li data-id="1">1</li>
                <li data-id="2">2</li>
                <li data-id="3">3</li>
            </ul>
 </div>
```

 ```CSS
 /*Css是使用父相子绝定位 子元素相对于父元素绝对定位*/
 /*要显示的一张图片大小*/
 .container{
  position:relative;
  width:500px;
  height:500px;
  overflow:hidden;/*超过部分隐藏*/
 }
 /*子容器添加所有的图片，横排排放*/
 .imglist{
  position:absolute;
  width:2500px;
  height:500px;
 }
 img{
  max-width:100%;/*防止图片变形*/
 }

 ```
