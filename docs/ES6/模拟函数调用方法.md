##### call方法
```javascript 1.8
var foo = {
    value: 1
};

function bar() {
    console.log(this.value);
}

bar.call(foo); //
```
* call 改变了this指向
* call 执行函数

##### 模拟call方法
* 将函数变成对象属性 指定了this
* 执行该函数
* 删除该函数属性
```javascript 1.8
//使用context获取函数调用对象
Function.prototype.call2=function(context,...args){
    context.fn=this
    context.fn(...args)
    delete context.fn //调用玩删除
}
```
